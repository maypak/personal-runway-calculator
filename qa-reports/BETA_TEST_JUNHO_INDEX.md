## 💬 대표 피드백 (Quote)

### 최준호의 핵심 메시지:
> **"계산기는 충분히 많아요.  
> 창업가에게 필요한 건 **시뮬레이터**입니다."**

### 현재 vs 필요:
```
현재 앱이 주는 것:
"런웨이: 5.6개월"

창업가가 원하는 것:
"지출 10% 줄이면 6.2개월 → 목표 달성 ✅
브릿지 ₩5M 확보 시 6.7개월 → 안전권 🎯
최악의 경우 4.6개월 → 위험 🔴"
```

### 사용 패턴 예상:

**현재 버전:**
- 온보딩 → "5.6개월" 확인 → 😰 타이트하네
- 시나리오 찾음 → 없음 → 스프레드시트로 이동
- **결과:** 재방문 안 함 (20%)

**개선 후:**
- 온보딩 → 목표 설정 → 시나리오 테스트
- 스크린샷 → 팀 공유 → 주간 리포트 활성화
- **결과:** 매주 체크 (80%)
