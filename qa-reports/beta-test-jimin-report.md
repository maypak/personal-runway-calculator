## 🎯 점수 평가

### 1. 사용성 (모바일): **8/10**

**잘된 점:**
- ✅ 터치 타겟 크기 충분 (`min-h-[44px]`, `min-w-[44px]`)
- ✅ 모바일 반응형 디자인 잘 구현됨 (`grid-cols-1 lg:grid-cols-3`)

- ✅ 진행 상황 표시 (Progress Dots) 명확
- ✅ 입력 필드에 플레이스홀더가 있어 이해하기 쉬움

**개선점:**
- ⚠️ 모바일에서 헤더 텍스트 잘림 가능성 (`truncate` 사용)

- ⚠️ 키보드 입력 시 숫자 키패드가 자동으로 나오지 않을 수 있음 (`inputMode="numeric"` 미사용)

## 😊 감정 반응 (최지민 관점)

### "3.2개월"을 보고 어떤 기분?

**솔직한 느낌:**
- 😰 **약간 불안해짐** - "생각보다 짧네... 3개월이면 여름까지밖에 못 버티잖아?"
- 😔 **현실 직시** - "알바 더 해야 하나? 아니면 대충 취업부터 해야 하나?"
- 🤔 **약간 안심됨** - "그래도 0개월은 아니니까... 3개월 안에 뭐라도 해야지"

### 격려 메시지가 위로됐나?

**"숨 쉴 여유가 있습니다. 지금이 계획을 세울 때입니다."**

- ✅ **긍정적:** "숨 쉴 여유가 있다"는 말이 약간 위로됨
- ⚠️ **부족함:** "계획을 세우라"는 게 추상적 - **구체적 행동 가이드가 필요**
- 💭 **더 원하는 것:** "친구들도 비슷할 거야" / "3개월이면 충분해" 같은 공감 메시지

### 앱 톤이 너무 무겁나, 적절한가?

- ✅ **적절함!** 진지하지만 너무 무겁지 않음
- 🎉 이모지 덕분에 가벼운 느낌 유지
- 💭 "You're doomed" 같은 무서운 메시지 없어서 다행

## 📱 모바일 UX 분석

### 좋았던 점 3가지:

1. **터치하기 편함** 
   - 버튼 크기가 충분함 (`min-h-[44px]`)
   - 카드 간격도 적당함 (`space-y-4`, `gap-4`)

2. **3단계 온보딩이 부담스럽지 않음**
   - 각 단계가 명확함
   - Progress Dots로 진행 상황 파악 쉬움
   - "다음 →" 버튼이 명확

3. **색상과 이모지로 시각적 피드백 즉각적**
   - 🟡 노란색 = 주의 필요
   - Progress Bar로 직관적 이해

### 개선 필요 3가지:

1. **숫자 입력이 불편함**
   - ❌ 천단위 구분 없음: "3800000" vs "₩3,800,000"
   - ❌ 숫자 키패드 자동 안 뜸 (`inputMode="numeric"` 필요)
   - 💡 **제안:** 슬라이더나 프리셋 버튼 추가 (예: "₩300만", "₩500만")

2. **"다음에 뭐 해야 하나?" 가이드 부족**
   - ❌ "3.2개월"을 보고 나서 뭘 해야 할지 막막함
   - 💡 **제안:** "당장 할 일" 체크리스트 추가
     - [ ] 알바 구하기
     - [ ] 지출 줄이기 (카페값 줄이기)
     - [ ] 실업급여 신청 가능한지 확인

3. **SNS 공유 기능 없음**
   - ❌ 인스타 스토리, 카톡으로 공유 버튼 없음
   - ❌ "내 친구는 5개월 버틴대" 같은 비교 기능 없음
   - 💡 **제안:** "친구들한테 공유하기" 버튼 추가

## 🚀 추가 원하는 기능

### 1. **더 많은 이모지 & 애니메이션** 🎉
- 런웨이 숫자가 나타날 때 카운팅 애니메이션
- 축하 이모지 폭발 효과 (12개월 이상일 때)
- 슬픈 이모지 (3개월 미만일 때 공감)

### 2. **SNS 공유 기능** 📲
- **카카오톡 공유:** "나 3개월 버틸 수 있대! 너도 해봐"
- **인스타 스토리:** 런웨이 결과 이미지 자동 생성
- **공유 문구 예시:**
  ```
  💸 나의 재정 런웨이: 3.2개월
  취준생 여러분, 우리 얼마나 버틸 수 있을까요?
  #런웨이계산기 #취준생 #재테크
  ```

### 3. **친구들과 비교 (익명)** 👥
- "같은 나이대 평균: 4.5개월"
- "구직자 평균: 5.2개월"
- "상위 25%: 8개월 이상"
- 💭 **익명 데이터 수집 필요** (동의 하에)

### 4. **위로 메시지 더 많이** 💬
- "3개월이면 충분해! 친구들도 비슷할 걸?"
- "알바 한 달만 하면 4.5개월로 늘어나!"
- "지출 10% 줄이면 3.5개월로!"

### 5. **What-if 시나리오** 🤔
- "알바 월 50만원 벌면?" → 4.2개월
- "카페값 월 10만원 줄이면?" → 3.5개월
- "부모님 도움 50만원 받으면?" → 5개월

### 6. **목표 설정 & 동기부여** 🎯
- "6개월까지 늘리기" 목표 설정
- 매일/매주 체크인 (푸시 알림)
- "3일째 지출 줄이기 성공! 🎉"

## 🐛 버그 리포트

### 모바일 UI 깨짐
- ❌ **헤더 텍스트 잘림:** `truncate` 사용으로 긴 제목 잘릴 수 있음
- 💡 **해결책:** 폰트 크기 조정 또는 줄바꿈 허용

### 터치 불편
- ⚠️ **숫자 키패드 안 뜸:** `inputMode="numeric"` 미사용
- 💡 **해결책:** 
  ```tsx
  <input
    type="text"
    inputMode="numeric"
    pattern="[0-9]*"
  />
  ```

### 오타
- ✅ **발견 안 됨** (코드 리뷰 기준)

### 이상한 표현
- ⚠️ **"wholemonths" 오타:** 
  ```ts
  const wholemonths = Math.floor(runwayMonths); // → wholeMonths
  ```
- ⚠️ **통화 포맷팅 개선 필요:**
  - ₩3.8M (영어권 표기)
  - ₩380만 (한국 선호 표기)
