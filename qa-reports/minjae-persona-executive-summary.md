## 🚨 긴급 개선 필요 (P0)

### 1. 보수적 계산 옵션 ⭐⭐⭐
```
현재: ₩30M ÷ ₩5M = 6.0개월 (100% 소진)
필요: 예비비 20% 옵션 → 4.8개월
```
**왜 필요**: 기혼자는 보수적 계산을 신뢰함

### 2. 위험 기준 명시 ⭐⭐⭐
```
🟢 안전: 9개월+ (가족 4인 기준)
🟡 주의: 6-9개월
🔴 경고: 3-6개월
🚨 위험: 3개월-
```
**왜 필요**: "6개월"이 안전한지 위험한지 모름

### 3. 가족 구성원 입력 ⭐⭐
```
독신: 6개월 충분
자녀 1명: 7개월 권장
자녀 2명: 9개월 권장
```
**왜 필요**: 가족 구성에 따라 안전 마진 다름
