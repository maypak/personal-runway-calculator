## 🚨 P0 - 출시 전 필수 (Critical)

### 1. 보수적 계산 옵션 추가
**문제**: "이 계산을 믿어도 되나?"  
**해결**: 예비비 옵션 제공

```javascript
// 현재
runway = assets / monthlyExpenses; // 6.0개월

// 제안
const conservativeMode = true;
const emergencyFundRate = 0.20; // 20%
const availableAssets = conservativeMode 
  ? assets * (1 - emergencyFundRate) 
  : assets;
runway = availableAssets / monthlyExpenses; // 4.8개월
```

**UI 제안**:
```
[ ] 보수적 계산 (예비비 20% 제외)
→ 체크 시: "4.8개월 (예비비 ₩6M 제외)"
→ 미체크 시: "6.0개월"
```

**임팩트**: 신뢰성 ↑↑ (5점 → 8점)

---

### 2. 위험 기준 명시
**문제**: "6개월이 안전한가, 위험한가?"  
**해결**: 색깔 + 수치 기준 제시

```javascript
function getRunwayStatus(months, familySize) {
  // 가족 크기에 따른 안전 마진 조정
  const safeThreshold = familySize >= 3 ? 9 : 6;
  const warningThreshold = safeThreshold * 0.67; // ~6개월 (가족 4인)
  const criticalThreshold = safeThreshold * 0.33; // ~3개월
  
  if (months >= safeThreshold) return { status: 'safe', color: 'green', icon: '🟢' };
  if (months >= warningThreshold) return { status: 'warning', color: 'yellow', icon: '🟡' };
  if (months >= criticalThreshold) return { status: 'critical', color: 'red', icon: '🔴' };
  return { status: 'danger', color: 'red', icon: '🚨' };
}
```

**UI 제안**:
```
민재 (가족 4인):
🟡 6.0개월 - 주의 단계
"가족이 있는 경우 최소 9개월 권장"

독신:
🟢 6.0개월 - 안전 단계
"독신의 경우 충분한 기간"
```

**임팩트**: 판단력 ↑↑, 불안 해소
