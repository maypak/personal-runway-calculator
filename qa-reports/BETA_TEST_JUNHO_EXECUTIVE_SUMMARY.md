## 🎯 핵심 인사이트

### ✅ 강점
1. **직관적 UI**: 3단계 온보딩, 30초 안에 결과 확인
2. **프라이버시**: 로컬 저장으로 신뢰 구축
3. **명확한 시각화**: 숫자 + 진행바 + 색상 코딩

### ❌ 약점 (창업가 관점)
1. **시나리오 분석 없음** ← **가장 치명적**
2. **목표일 설정 없음** (예: "6개월 후 투자 유치")
3. **액션 플랜 부재** ("무엇을 해야 하는가?")

---

## 💡 핵심 발견

### 창업가의 질문:
> "6개월 안에 투자 받아야 하는데 가능한가?"

### 현재 앱의 답변:
> "런웨이: 5.6개월"

### 창업가가 원하는 답변:
```
⚠️ 타이트: 목표까지 0.4개월 부족

시나리오:
- 지출 10% 절감 → 6.2개월 ✅ 목표 달성
- 브릿지 ₩5M → 6.7개월 🎯 안전권
- 최악 (+20%) → 4.6개월 🔴 위험

💡 제안: 지출 ₩450,000 절감 또는 브릿지 펀딩 검토
```

---

## 🚨 P0 개선 요청 (출시 전 필수)

### 1. 시나리오 비교 기능
**필요성:** 창업가는 "What-if" 질문에 답이 필요
**구현:**
- 슬라이더로 지출 ±30% 조정
- 실시간 런웨이 변화 표시
- 3가지 시나리오 자동 생성 (절약/현재/최악)

**예상 효과:**
- 만족도 6/10 → 9/10
- 재방문율 20% → 80%

### 2. 목표일 설정 & 역산
**필요성:** "6개월 후"라는 목표가 있을 때 역산 필요
**구현:**
- 목표일 선택 UI
- 현재 vs 목표 비교
- 부족분 자동 계산

**예상 효과:**
- 의사결정 속도 향상
- 액션 플랜 명확화

### 3. 데이터 수정 플로우
**필요성:** 온보딩 후 수정 방법 불명확
**구현:**
- Dashboard에 "설정 수정" 버튼
- 단계별 수정 가능

## 📈 P1 개선 제안 (출시 후 1개월)

1. **지출 상세 분석**
   - 카테고리별 입력 (사무실/인건비/마케팅)
   - 절감 가능 항목 표시

2. **스타트업 특화 지표**
   - MRR (월간 반복 수익)
   - Burn rate 트렌드
   - CAC (고객획득비용)

3. **주간 리포트**
   - 런웨이 변화 추적
   - 경고 알림 (2개월 남음 등)

## 🎭 페르소나 검증

### 최준호 (창업가)의 현실
- **자산:** ₩25,000,000
- **월 지출:** ₩4,500,000 (사무실 포함)
- **월 수입:** ₩0
- **런웨이:** 5.56개월
- **목표:** 6개월 안에 투자 유치
- **상태:** ⚠️ 타이트 (margin 거의 없음)

### 앱 사용 패턴 예상

**현재 버전:**
1. 온보딩 완료 (2분)
2. "5.6개월" 확인
3. "타이트하네..." 😰
4. 시나리오 기능 찾음 → 없음
5. 스프레드시트 열어서 수동 계산
6. **결과:** 재방문 안 함

**개선 후 버전:**
1. 온보딩 완료 (2분)
2. 목표일 설정: 6개월 후
3. 시나리오 테스트: 지출 -10% → ✅ 가능
4. 스크린샷 → 공동창업자 공유
5. 주간 리포트 활성화
6. **결과:** 매주 체크, 투자자 미팅 때 활용
