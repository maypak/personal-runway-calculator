## 🟡 P1 - 베타 중 개선 필요

### 5. 이메일 변경 불가 ❌
**현상:** Supabase auth.users.email 변경 기능 없음
**영향:** 오타로 가입한 유저가 수정 불가
**해결:** Supabase `updateUser()` 사용
**우선순위:** **중간** (베타 중 추가 가능)

### 6. 자동 저장 명시 안 됨
**현상:** Settings 변경 시 "저장됨" 표시 없음
**영향:** 유저가 저장됐는지 불안해함
**해결:** 임시 "✓ Saved" 토스트 메시지
**우선순위:** **낮음** (UX 개선)

### 7. 오프라인 처리 없음 ❌
**현상:** 네트워크 끊김 시 에러 메시지만
**영향:** 유저 혼란, 데이터 손실 위험
**해결:** "You're offline" 배너 + 재연결 시 자동 동기화
**우선순위:** **낮음** (PWA 구현 시 함께)

**우선순위:** **EU 타겟 시 필수**

**현상:** 홈 화면 추가, 오프라인 작동 불가
**영향:** 모바일 사용성 저하
**해결:** Service Worker + manifest.json 개선
**우선순위:** **정식 런칭 전**
