# UX & Service Audit Checklist
## 사용자 경험 + 서비스 기본 기능 종합 점검표

> **목적:** 우리가 "못해서"가 아니라 "점검해야 하는지 몰라서" 놓친 것들을 찾기 위한 체크리스트

---

## 📋 사용 방법

**베타 런칭 전:** 모든 항목 점검
**정기 점검:** 매주 월요일 + 주요 기능 추가 후
**신규 기능:** 해당 섹션만 집중 점검

**우선순위:**
- 🔴 P0: 런칭 전 필수
- 🟡 P1: 베타 중 개선
- 🟢 P2: 정식 런칭 전

---

## 1️⃣ 신규 사용자 여정 (First-Time User Experience)

### 1.1 첫 방문 (Landing/Auth)
- [ ] 🔴 가치 제안이 5초 안에 이해되는가? ("Your money is TIME")
- [ ] 🔴 회원가입 버튼이 명확한가?
- [ ] 🔴 소셜 로그인이 작동하는가? (Google, GitHub)
- [ ] 🔴 비밀번호 조건이 명확한가? (현재: ✅)
- [ ] 🟡 로딩 상태가 표시되는가?
- [ ] 🟡 에러 메시지가 도움이 되는가?
- [ ] 🟢 "왜 가입해야 하는가?" 설득력 있는가?

### 1.2 첫 로그인 후 (Onboarding)
- [ ] 🔴 빈 대시보드에 던져지지 않는가? (현재: ✅ Onboarding Wizard)
- [ ] 🔴 첫 번째 "의미 있는 결과"까지 60초 이내인가? (현재: ✅)
- [ ] 🔴 다음 단계가 명확한가? ("Get Started" 버튼)
- [ ] 🟡 건너뛰기 옵션이 있는가? (현재: ✅)
- [ ] 🟡 진행률이 표시되는가? (현재: ✅ "Step 2 of 4")
- [ ] 🟡 축하/보상 느낌이 있는가? (현재: ✅ Confetti)
- [ ] 🟢 튜토리얼/가이드 투어가 있는가?

### 1.3 첫 데이터 입력
- [ ] 🔴 입력 필드가 이해하기 쉬운가?
- [ ] 🔴 플레이스홀더가 예시를 보여주는가? (현재: ✅ "45000")
- [ ] 🔴 검증 에러가 즉시 표시되는가?
- [ ] 🟡 자동 저장되는가? (새로고침 시 데이터 유지)
- [ ] 🟡 입력 도움말(툴팁)이 있는가? (현재: ✅)
- [ ] 🟢 샘플 데이터 자동 채우기 옵션이 있는가?

### 5.2 스크린 리더
