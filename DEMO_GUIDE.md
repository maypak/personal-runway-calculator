# 🎯 Personal Runway Calculator - Demo Guide

**Quick Start:** 30초 안에 런웨이 계산 완료!

---

## 🚀 실행 방법

### Development Mode
```bash
npm run dev
```
- Open: http://localhost:3000
- Hot reload 활성화

### Production Build
```bash
npm run build
npm start
```

## 📱 사용자 플로우 (Happy Path)

### 1. 첫 방문 (/)
- **자동 리다이렉트** → `/onboarding` (데이터 없을 경우)
- 로딩 화면 표시 (스피너)

### 2. 온보딩 플로우 (/onboarding)

#### Step 1: 상황 선택
```
┌──────────────────────────────┐
│ 당신의 상황을 선택해주세요     │
│                              │
│ 💼 프리랜서                   │
│ 불규칙한 수입을 관리하고...   │
│                              │
│ 🔍 구직자                     │
│ 제한된 예산으로 얼마나...     │
│                              │
│ 🚀 창업가                     │
│ 번아웃 전에 런웨이를...       │
│                              │
│ ⚡ 빠른 계산                  │
│ 간단하게 런웨이만...          │
└──────────────────────────────┘
```
- **액션**: 카드 클릭 → 자동으로 Step 2 이동

#### Step 2: 자산 입력
```
┌──────────────────────────────┐
│ 💰 현재 가지고 계신 자산은?   │
│                              │
│    ₩ 2,500,000              │
│    [입력 필드]               │
│                              │
│ 💡 포함: 은행 계좌, 비상금... │
│                              │
│ [← 이전]         [다음 →]   │
└──────────────────────────────┘
```
- **입력**: 숫자만 입력 가능
- **검증**: 0보다 커야 다음 버튼 활성화
- **키보드**: Enter → 다음

#### Step 3: 월 지출 입력
```
┌──────────────────────────────┐
│ 📊 월 평균 지출은 얼마인가요? │
│                              │
│    ₩ 850,000                │
│    [입력 필드]               │
│                              │
│ 💡 일반적: ₩2.5M-5M/월       │
│                              │
│ [← 이전]     [계산! 🎯]     │
└──────────────────────────────┘
```
- **입력**: 월 평균 지출 (숫자)
- **옵션**: 변동 소득 토글 (프리랜서 선택 시)
- **완료**: "계산!" 버튼 → Dashboard로 이동

## 🧪 테스트 시나리오

### Scenario A: 긴급 상황 (Critical)
```
자산: ₩2,000,000
월 지출: ₩1,000,000
→ 런웨이: 2.0개월
→ 색상: 🔴 빨강
→ 종료일: 2026년 4월 23일
```

### Scenario B: 경고 (Warning)
```
자산: ₩5,000,000
월 지출: ₩1,000,000
→ 런웨이: 5.0개월
→ 색상: 🟡 노랑
→ 종료일: 2026년 7월 23일
```

### Scenario C: 양호 (Good)
```
자산: ₩10,000,000
월 지출: ₩1,000,000
→ 런웨이: 10.0개월
→ 색상: 🟢 초록
→ 종료일: 2026년 12월 23일
```

### Scenario D: 우수 (Excellent)
```
자산: ₩20,000,000
월 지출: ₩1,000,000
→ 런웨이: 20.0개월
→ 색상: 🔵 파랑
→ 종료일: 2027년 10월 23일
```

### Scenario E: 무한 (Infinite)
```
자산: ₩10,000,000
월 지출: ₩1,000,000
월 수입: ₩1,500,000
→ 런웨이: 무한 (수입 > 지출)
→ 색상: 🔵 파랑
```
