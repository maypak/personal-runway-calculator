# Personal Runway Calculator - 한국어 설명 자료

**만든 이유:** 10년 다니던 회사 그만두기 전에 제일 무서웠던 게 "돈 얼마나 버틸 수 있지?" 였습니다.  
**해결책:** 돈을 '시간'으로 바꿔서 보여주는 계산기를 만들었습니다.

---

## 🎯 핵심 컨셉 (30초 설명)

> **"돈은 돈이 아니라 시간입니다"**

**예시:**
- ❌ "저축 5천만원 있어요" → 뭔가 추상적
- ✅ "2년 7개월 버틸 수 있어요" → 명확함!

**이게 Personal Runway입니다.**

---

## 💡 누가 쓰나요?

### 1. 퇴사 준비하는 직장인 (30-40대)
- "회사 그만두고 싶은데 얼마나 버틸까?"
- "창업하려면 런웨이가 얼마나 필요할까?"

### 2. 프리랜서/사이드 프로젝트 하는 사람
- "수입이 들쭉날쭉한데 언제까지 가능할까?"
- "본업 그만두고 프리랜서 전업할까?"

### 3. FIRE (조기은퇴) 준비하는 사람
- "지금 은퇴하면 몇 년 버틸까?"
- "연 4% 룰 말고 실제 내 상황으로 계산하고 싶어"

---

## 🚀 뭘 할 수 있나요?

### 1️⃣ 런웨이 계산 (30초)
```
저축: 5천만원
고정비: 150만원/월
변동비: 30만원/월

→ 결과: 2년 7개월 버틸 수 있음
```

### 2️⃣ 지출 추적
- 오늘 쓴 돈 기록 → 런웨이 실시간 업데이트
- "커피 5천원 샀더니 런웨이가 1시간 줄었네?"

### 3️⃣ What-if 시뮬레이션
- "지출 30% 줄이면?" → 런웨이 3년 5개월
- "프리랜서로 월 200 벌면?" → 런웨이 5년 2개월

### 4️⃣ 모바일 친화적
- 스마트폰에 설치 (PWA)
- 5초 안에 런웨이 확인 가능

## 🤔 자주 묻는 질문

### Q: 엑셀로 하면 되지 않나요?
**A:** 맞습니다! 하지만 엑셀은:
- 복잡함 (수식 짜야 함)
- 모바일에서 쓰기 불편
- 매번 열어야 함

Personal Runway는:
- 30초 안에 계산
- 스마트폰에서 5초 만에 확인
- 매일 쓰기 쉬움

### Q: YNAB, Mint 같은 앱이랑 뭐가 다른가요?
**A:** 
- YNAB/Mint: 예산 관리, 자산 추적
- Personal Runway: **런웨이 계산**

목적이 달라요. "몇 개월 버틸까?"에 집중합니다.

### Q: 정말 무료인가요? 나중에 유료 전환 아닌가요?
**A:** 
- 기본 기능은 영원히 무료
- 나중에 프리미엄 기능 추가될 수 있음 (예: 멀티 통화, PDF 내보내기)
- 하지만 지금 쓰는 기능은 절대 유료화 안 함

### Q: 데이터 안전한가요?
**A:** 
- Supabase (수천 개 회사가 쓰는 서비스) 사용
- 암호화 (전송 중, 저장 시)
- 오픈소스 (코드 검증 가능)
- 원하면 직접 호스팅도 가능 (GitHub 코드 공개)
